# Dash Saturn

**Dash Saturn** is a modern Saturn-compatible desktop launcher for legally dumped games.

It helps users organize their Saturn-compatible game library, manage cover artwork, configure BIOS paths, handle multi-disc titles, and launch games through a clean desktop interface.

> Independent project. No BIOS files, games, ROMs, disc images, or copyrighted firmware are included.

## Features

- Modern game library
- Cover artwork support
- BIOS path management
- Multi-disc game support
- `.m3u` playlist handling
- Runtime launch integration
- Save and state organization
- Clean desktop interface

## Project Status
